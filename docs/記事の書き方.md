# 記事の書き方（見出し・色・ハイライト）

WordPress のように「H2＋本文」の構造や、**文字色・ハイライト**を使えます。管理画面の「本文」は **Markdown** で書き、必要に応じて **HTML** を混ぜられます。

---

## 1. 見出し（H2・H3）と本文の構造

Markdown で次のように書くと、見出しと本文の構造になります。

```markdown
## はじめに

ここが「はじめに」の本文です。H2の下に普通に文章を書けばOKです。

## 学校職員の仕事内容

この見出しの下も本文です。

### 事務職の役割

H3はさらに細かい見出しです。目次にも出ます。
```

- **`##`** → 大見出し（H2）
- **`###`** → 小見出し（H3）
- その下に普通に文章を書く → 本文

※ 公開ページでは目次（TOC）が自動で作られ、H2・H3が一覧になります。

---

## 2. ハイライト（マーカー風）

**黄色っぽいマーカー**をつけたい部分は、次のどちらかで囲みます。

**方法A：`<mark>` タグ**
```html
<mark>ここがハイライトされます</mark>
```

**方法B：`highlight` クラス**
```html
<span class="highlight">ここもハイライトされます</span>
```

記事本文にそのまま貼り付けて保存すれば、表示でマーカーが付きます。

---

## 3. 文字の色

**アクセント色（赤系）** をつけたいとき：
```html
<span class="text-accent">重要なキーワード</span>
```

**グレー（補足っぽく）** にしたいとき：
```html
<span class="text-muted">補足の説明</span>
```

---

## 4. 使用例（コピー用）

```markdown
## 結論

学校職員は<mark>採用試験の合格</mark>が第一歩です。

<span class="text-accent">公募採用</span>と<span class="text-accent">公務員試験</span>の両方があります。

## 次のステップ

<span class="text-muted">詳細は各自治体のHPをご確認ください。</span>
```

- 「採用試験の合格」→ ハイライト
- 「公募採用」「公務員試験」→ アクセント色
- 最後の一文 → グレー

---

## 5. 注意

- 本文は **Markdown** が基本です。見出しは `##` / `###` で書くのがおすすめです。
- 色・ハイライトは **HTML**（`<mark>`, `<span class="...">`）をそのまま本文に書いて使います。
- 管理画面の「テンプレ挿入」で、H2＋本文のひな形が入るので、その中に上記のタグを追加しても大丈夫です。
